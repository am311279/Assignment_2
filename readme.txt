Project Description:
--------------------
This is a responsive User Management interface built with HTML5 and Bootstrap 5. It features a modern navigation bar, a user input form with built-in validation, a table for displaying user data, and image and button utility sections to showcase Bootstrap styling.

Features:
---------
1. Navigation Bar:
   - Fixed to the top
   - Responsive with collapsible menu
   - Styled using Bootstrap's `navbar-dark` and `bg-primary` classes

2. User Form:
   - Collects First Name, Last Name, Email, and Password
   - Email and Password fields use Bootstrap validation classes
   - Includes checkbox for Terms Agreement
   - Form styled with Bootstrap grid, spacing, and shadow utilities

3. User Table:
   - Displays sample user data (Name, Email, Role)
   - Responsive, striped, and hover-enabled table using Bootstrap classes

4. Images:
   - Responsive full-width image
   - Rounded profile-style circular image

5. Buttons:
   - Primary and Warning buttons with responsive visibility classes
   - Center-aligned using Bootstrap's flexbox and grid utilities

Technologies Used:
------------------
- HTML5
- Bootstrap 5.3 (via CDN)

Folder Structure:
-----------------
- index.html             → Main HTML file 
- Image/Image1.jpg       → Image file used in the page (ensure this path exists)

Setup Instructions:
-------------------
1. Download or clone this project folder.
2. Ensure the image file `Image1.jpg` is placed inside an `Image/` folder relative to the HTML file.
3. Open the HTML file in any modern web browser (e.g., Chrome, Firefox, Edge).
4. No additional setup or dependencies are required since Bootstrap is loaded via CDN.
